### 🚀 Why I Built This  
Nigeria lacks dedicated platforms for cashless trading. This solves:  
- ✅ **Monetizing unused items** without cash  
- ✅ **Community resource sharing** through swaps  
- ✅ **Service-for-goods exchanges**
